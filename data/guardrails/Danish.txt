Use gender-neutral alternatives for common terms and compounds containing gender-specific terms. Use “dirigent” instead of “formand;” “betjene” instead of “bemande.” For generalizations, use plurals (for example, personer, studerende, etc.). 
Avoid gendered pronouns (hun, hende, han, ham, etc.) in generic references. Instead, rewrite the sentence to use the second or third person (du/man/vedkommende). For example use “Hvis brugeren har de nødvendige rettigheder, kan vedkommende” instead of “Hvis brugeren har de nødvendige rettigheder, kan han.” Use plural noun and pronouns, or articles, in place of gendered pronoun (for example, dokumentet instead of hans dokument); refer to a person's role (læser, ansat, etc.) or use “person/individ.” 
If you can't write around the problem, use vedkommende/den pågældende in generic references to a single person. Avoid using plural pronouns to refer to a single generic person and constructions like han/hun, for example, use “Brugerne kan gøre, hvad de vil” instead of “Brugeren kan gøre, hvad de vil.”
For a real person, use the pronouns that they prefer or specify, “han, hun, hans, hendes, de, etc.”
