Use gender-neutral alternatives for common terms and compounds containing gender-specific terms. For generalizations, use plural nouns “personen, menschen,” etc. 
Avoid gendered pronouns in generic references. Instead, rewrite the sentence to use the second or third person (Sie); use plural noun and pronoun; use articles like, “das Dokument” instead of “sein/ihr Dokument;” or, address the subject directly like “bitte unterschreiben Sie das Formular“ instead of “der Antragssteller muss das Formular unterschreiben.” When possible, use suffixes “-hilfe, -kraft, -personal, -schaft, -leute” to replace gendered forms. 
Replace gendered nouns like “(Abstand) nach vorne; (Abstand) zur Person vor sich” with “(Abstand) zum Vordermann;” or, “Bankkaufperson” instead “Bankkaufmann.” If a gender-neutral noun exists, it should be used.  If there is no singular or plural gender-neutral noun, the gender asterisk should be used for both forms. For example, use “Teilnehmer*in (singular),” “Teilnehmende (plural),” instead of “Teilnehmer.” Example sentence: use “Wenn Autor*innen Dokumente öffnen...” instead of “Wenn der Autor sein Dokument öffnet...” 
For a real person, use the pronouns that they prefer or specify, like “er, sie, sein, ihr” or another pronoun.
